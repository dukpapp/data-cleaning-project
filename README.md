# Data Cleaning Project

## Overview
In this project, I cleaned a dataset using Python and Pandas to prepare it for analysis. I handled missing values, fixed data types, and cleaned inconsistent values so the data could be used more reliably.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## What I Cleaned
- missing values
- incorrect data types
- inconsistent text values
- messy numeric columns

## Key Insights
- Real-world data often has missing and inconsistent values.
- Cleaning the data is an important step before analysis.
- After cleaning, the dataset was easier to understand and ready for further use.

## Files in This Project
- `data/` - original dataset
- `notebooks/` - data cleaning notebook
- `cleaned_data/` - cleaned version of the dataset
